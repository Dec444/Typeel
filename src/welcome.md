# Welcome to Typeel

A calm, **free** place to write — type, then organize your words and ideas layer by layer, like peeling an onion until the clearest thought is left.

## Getting started

- Press **New** (or `Ctrl / Cmd + N`) to start a fresh document
- Press **Open** (or `Ctrl / Cmd + O`) to open a `.md` file, and **Save** with `Ctrl / Cmd + S`
- Click the folder icon in the sidebar to browse a project — its name and files appear on the left
- This **Welcome** note always sits in the sidebar, so you can open it any time for a refresher

## A few markdown tricks

Type `/` on a new line for the block menu, or just write:

- **bold**, *italic*, and `inline code`
- lists, like this one
- tables, quotes, and fenced code blocks

> Good writing starts messy. Get it all down, then peel it back until the clearest thought is left.
